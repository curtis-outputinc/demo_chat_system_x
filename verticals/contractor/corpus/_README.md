folder onto the system prompt at runtime (excluding files starting
with `_` or `.`, so this README is ignored).

See `verticals/injury-lawyer/corpus/` for the canonical shape of
how each file is structured (clear headers, short sections, plain
prose, no markdown bullets in the answer text itself).
